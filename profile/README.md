## Welcome to Golden Linux!
This is the official GitHub organization for **Golden Linux**, a new beginner-friendly Linux distribution.

### How is this different?
Golden Linux started when I ([AnIDFB](https://github.com/IdfbAn)) noticed that there was no beginner-friendly distro quite up to my ridiculously high standards.
So what are those standards, and thus the goals of Golden Linux?
* Built to be easily usable by beginners (duh!)
* Uses modern technologies, Wayland by default in particular is important to me
* Not created by a corporation, nor based on a distro created by a corporation
* Software is up to date
* This is a more minor one, but a graphical package manager is also nice

### The plan
At this moment in time, nothing is actually here yet. The plan is to create these in the following order:
1. The packages, using a simple metadata + script structure
2. The package manager, though most people will use KDE Discover rather than it directly
3. The live image builder
4. The installer, maybe a modified Calamares?

Golden Linux will most likely only have KDE Plasma for a desktop environment, due to its popularity and similarity to Windows,
as well as to prevent analysis paralysis and stretching myself too thin by having both it and GNOME.
